# HeartShield: Early Heart Disease Risk Prediction

A web application that predicts the risk of heart disease based on patient data and medical documents.

## Features

- User-friendly web interface
- Manual data input
- Document upload (PDF, JPG, PNG) with OCR
- Heart disease risk prediction
- Detailed results and recommendations
- Responsive design
